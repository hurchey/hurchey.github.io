Project 1

Name Tag

(beginniner 3D model)

Part 1

  • Although it seems very easy to do, it was actually quite a challenge to ensure that the other parts of this print will all in check such as the 3D printer working properly, the file that was downloaded into the 3D printer, and understanding how to use PrusaSlicer.
  • I am missing some photos of my previous attempts at making my name tag, but this came out to be my final product that very much like and happy about.
  • To create as such, I first went into the PrusaSlicer software on my laptop so that I first edit the 3D render of my nametag to have my name on it. Then I had to download that into a G-Code file that the 3D printer could read. Then I downloaded that G-Code file into a flash drive which was then put into the 3D printer so I could then select the file to be printed. It was about a 15 minute print from start to finish. I used one filament color for the whole name tag.

Part 2 Incorrect Failure 2

(failure in using the software correctly)

Part 2

  • Exact same steps as making the Name Tag print.
  • Despite this looking like nothing failed in terms of printing, this was supposed to be a test to see how 3D printers would react when you do not use the software correctly. My hope for this print was to test the 3D printer on how it would draw the ears of this keychain without any support beams so that in theory, the 3D printer would start to draw the ears from the surface of the 3D board rather than where the ears should be. While I was playing around with the PrusaSlicer software and trying to remove all of the support beams on the ears, it seems as if it did not happen so the whole print went through with no errors which is considered a failure in this context.
  • This print took a total of 3.5 hours.

Part 2 Correct Failure

(stress testing 3D printers)

Part 2

  • Exact same steps as making the Name Tag print.
  • This print is supposed to replace the Stitch Keychain print that I did before since I wanted to test to see what would happen to the 3D printer if there was something suspended in mid air with no support beams to help it print. I realize that without support beams when trying to print something like a bridge or ears that are in the air, the 3D printer will just start to print on the surface of the 3D printing board and think that is where it left off causing the filament to be all over the place and not stable at all and very weak. I realize that support beams are extremely important and needed in almost every kind of build that are similar to the ones that I did for this project.
  • This print took a total of 10 minutes.

Cup (Initial Creation)

(practice remixing 3D files using Fusion)

Part 3

  • So based on my last creation, I decided to change up and create something from Fusion which was a cup. It was a bit of learning curve to try to learn Fusion and implement things in Fusion, but once I got the hang of it, I was able to make a cup 3D model. I was using Fusion's lines and different shapes that are already in the software and try to make the cup. It was pretty difficult having everything line up as how a cup should be.
  • This is where my problem happened, the model that I created had a very major flaw in that the bottom of the cup was not even and PrusaSlicer could not read my print to be flat on the printing board which was very weird.
  • Even after many attempts to try to fix my issue and trying to make the cup as smooth and straight as possible, nothing seemed to work. I even tried to print it with support beams to see if that would fix or do anything. While I do not have a photo of what happened due to my forgetfulness, the print from the first 3 minutes already failed and did not stick to the board. This part of the project also seemed like a failure.
